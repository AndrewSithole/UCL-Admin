Basic Checkbox

You can use the Checkbox component to render a checkbox input.

Checkbox Colors

Use the color prop to change the color of the checkbox. You can use the following colors: primary, secondary, success, danger, warning, info, dark. Default color is primary.

Checkbox Sizes

Use the size prop to change the size of the checkbox. You can use the following sizes: sm, md, lg. Default size is sm.
