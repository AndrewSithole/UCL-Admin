Button Colors

Use color class to btn-primary-solid, btn-secondary-solid, btn-success-solid, btn-danger-solid, btn-warning-solid btn-info-solid & btn-dark-solid.

Button Sizes

Use size class to btn-xs, btn-sm, btn-md, btn-lg & btn-xl Default Value is btn-md.

Outline Buttons

Use outline class to btn-primary-outline, btn-secondary-outline, btn-success-outline, btn-danger-outline, btn-warning-outline, btn-info-outline & btn-dark-outline create a outlined button.

Soft Buttons

use soft class to btn-primary-soft, btn-secondary-soft, btn-success-soft, btn-danger-soft, btn-warning-soft, btn-info-soft & btn-info-soft create a soft button.

Plain Buttons

Use plain class to btn-primary-plain, btn-secondary-plain, btn-success-plain, btn-danger-plain, btn-warning-plain, btn-info-plain & btn-dark-plain create a soft button.

Rounded Buttons

Userounded-full class to create a rounded button.

Icon Buttons

By adding icon to the button, you can create icon buttons

Loading Buttons

By adding a loading icon with disabledclass, you can create loading buttons.

Disabled Buttons

Use disabled, class to create a disable button.

React Icons
Block Buttons

Use w-full, class to create block button.

Group Buttons

Use ButtonGroup class to create a group of buttons.
